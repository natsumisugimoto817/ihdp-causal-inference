# IHDP Causal Inference

## 1. Overview

このプロジェクトでは、**Infant Health and Development Program（IHDP）** のデータを用いて、乳幼児向け介入プログラムが認知スコアに与える効果を推定します。

目的は、介入プログラムへの参加が、乳幼児の認知スキルスコアにどの程度影響を与えたのかを推定することです。

本プロジェクトでは、以下のように変数を定義します。

* `T`: 介入の有無を表す処置変数

  * `T = 1`: 介入プログラムを受けた
  * `T = 0`: 介入プログラムを受けていない
* `Y`: アウトカム変数。認知スキルスコアを表す
* `X`: 観測された共変量・交絡因子。乳幼児や家庭に関する特徴量を表す（X_1からX_25まで）

本プロジェクトでは、以下の3つの因果推論手法を比較します。

1. 傾向スコアマッチング
2. 逆確率重みづけ推定
3. 線形回帰による調整

主に推定したい対象は、**平均処置効果（Average Treatment Effect: ATE）** です。

```text
ATE = E[Y(1) - Y(0)]
```

---

## 2. Problem Setting

本分析の目的は、介入プログラム `T` が認知スコア `Y` に与える効果を推定することです。

因果推論では、各個体に対して2つの潜在アウトカムを考えます。

```text
Y(1): 介入を受けた場合のアウトカム
Y(0): 介入を受けなかった場合のアウトカム
```

個体ごとの因果効果は、以下のように表せます。

```text
Y(1) - Y(0)
```

しかし、同じ乳幼児について、`Y(1)` と `Y(0)` を同時に観測することはできません。

ある乳幼児は、介入を受けるか、受けないかのどちらか一方だからです。

これを、因果推論の根本問題と呼びます。

そのため、本プロジェクトでは、個体ごとの因果効果を直接観測するのではなく、集団全体における平均的な因果効果を推定します。

```text
ATE = E[Y(1) - Y(0)]
```

---

## 3. Why Simple Comparison Is Not Enough

単純に介入群と非介入群の平均アウトカムを比較すると、以下のようになります。

```text
E[Y | T = 1] - E[Y | T = 0]
```

しかし、この差がそのまま介入プログラムの真の効果を表すとは限りません。

理由は、介入の割り当て `T` が、観測された交絡因子 `X` の影響を受けている可能性があるためです。

例えば、低出生体重の乳幼児や発達リスクが高い乳幼児ほど、介入プログラムを受けやすかった可能性があります。

その場合、介入群と非介入群は、もともとの特徴が異なります。

したがって、認知スコアの差は、以下のどちらによるものか区別できません。

* 介入プログラムそのものの効果
* もともとの乳幼児や家庭の特徴の違い
* その両方

そのため、本プロジェクトでは、観測された交絡因子を調整したうえで、介入効果を推定します。

---

## 4. Data

本プロジェクトでは、IHDPデータセットを使用します。

このデータセットには、介入の有無、アウトカム、複数の共変量が含まれています。

| Variable     | Description     |
| ------------ | --------------- |
| `T`          | 介入の有無を表す処置変数    |
| `Y`          | 認知スコアを表すアウトカム変数 |
| `X_1 ~ X_25` | 観測された共変量・交絡因子   |

介入効果を推定する前に、以下の観点でデータを確認します。

1. 介入群と非介入群のサンプル数
2. アウトカム変数 `Y` の分布
3. 共変量 `X` と介入の有無 `T` の関係
4. 共変量 `X` とアウトカム `Y` の関係
5. 介入群と非介入群の共変量バランス

このステップの目的は、調整前の介入群と非介入群がどの程度比較可能かを確認することです。

---

## 5. Methods

本プロジェクトでは、以下の3つの方法で介入効果を推定します。

1. 傾向スコアマッチング
2. 逆確率重みづけ推定
3. 線形回帰による調整

いずれの方法も、観測された交絡因子を調整することを目的としています。

ただし、それぞれの方法は、考え方や前提が異なります。

---

## 5.1 Propensity Score Matching

### Concept

傾向スコアマッチングでは、介入を受けた個体と、似た特徴を持つ非介入の個体を比較します。

ただし、複数の共変量をそのまま使って完全に似た個体を探すのは難しいため、共変量を1つのスコアに要約します。

そのスコアが、**傾向スコア** です。

傾向スコアは、観測された共変量 `X` のもとで、介入を受ける確率を表します。

```text
e(X) = P(T = 1 | X)
```

本プロジェクトでは、ロジスティック回帰などのモデルを用いて傾向スコアを推定します。

直感的には、以下のような考え方です。

```text
介入を受ける確率が近い個体同士を比較する
```

### Procedure

1. 観測された共変量 `X` を使って、傾向スコア `e(X)` を推定する
2. 介入群の各サンプルに対して、傾向スコアが近い非介入群のサンプルを対応させる
3. マッチング後の介入群と非介入群で、アウトカム `Y` の平均差を比較する
4. マッチング後に共変量バランスが改善したかを確認する

### Balance Check

マッチングがうまくできたかを確認するために、**標準化平均差（Standardized Mean Difference: SMD）** を用います。

SMDは、介入群と非介入群の共変量の差を標準化して表す指標です。

平均差が標準偏差の何個分ズレているか
SMD = (mean_treated - mean_control) / pooled_sd

pooled_sd（プールした標準偏差）は、介入群と非介入群の標準偏差をまとめたもの
pooled_sd = sqrt((sd_treated^2 + sd_control^2) / 2)

マッチング後にSMDが小さくなっていれば、介入群と非介入群の共変量バランスが改善したと考えられます。

## 5.2 Inverse Probability Weighting

### Concept

逆確率重みづけ推定では、傾向スコアを使って各サンプルに重みをつけます。

重みは以下のように定義します。

```text
介入群の場合:
weight = 1 / e(X)

非介入群の場合:
weight = 1 / (1 - e(X))
```

ここで、`e(X)` は傾向スコアです。

直感的には、それぞれの群の中で「珍しい特徴を持つサンプル」に大きな重みをつける方法です。

例えば、ある個体の傾向スコアが低いにもかかわらず、実際には介入を受けていた場合、その個体は介入群の中では珍しいサンプルです。

そのため、このような個体には大きな重みをつけます。

この重みづけによって、介入の有無が観測された共変量にあまり依存しないような、疑似的な母集団を作ることを目指します。

### Procedure

1. 傾向スコア `e(X)` を推定する
2. 各サンプルに逆確率重みを計算する
3. 介入群の重み付き平均アウトカムを計算する
4. 非介入群の重み付き平均アウトカムを計算する
5. 両者の差をATEとして推定する

IPWによるATE推定量は、以下のように表せます。

```text
ATE_IPW = E[T * Y / e(X)] - E[(1 - T) * Y / (1 - e(X))]
```

### Notes

IPWは、傾向スコアが極端な値をとる場合に不安定になりやすいです。

例えば、`e(X)` が0や1に近い場合、重みが非常に大きくなります。

そのため、本プロジェクトでは以下を確認します。

1. 傾向スコアの分布
2. 介入群と非介入群の共通サポート
3. 重みの分布
4. 重みづけ後の共変量バランス

必要に応じて、重みのトリミングや安定化重みの使用も検討します。

---

## 5.3 Linear Regression Adjustment

### Concept

線形回帰による調整では、アウトカム `Y` を、介入変数 `T` と観測された共変量 `X` によって説明するモデルを作ります。

モデルは以下のように表せます。

```text
Y = β0 + β1T + β2X1 + ... + βkXk + ε
```

このモデルにおいて、`T` の係数である `β1` を、観測された共変量を調整したうえでの介入効果として解釈します。

```text
ATE ≈ β1
```

直感的には、以下のような考え方です。

```text
観測された共変量の影響を固定したうえで、介入の効果を見る
```

### Procedure

1. `Y` を目的変数にする
2. `T` と共変量 `X` を説明変数にする
3. 線形回帰モデルを推定する
4. `T` の係数を、調整後の介入効果として解釈する

### Notes

この方法は、回帰モデルの指定に依存します。

例えば、共変量とアウトカムの関係が非線形である場合、単純な線形モデルでは交絡を十分に調整できない可能性があります。

追加分析として、マッチング後のデータに対して線形回帰を行い、残った共変量の差をさらに調整することも検討します。

## 6. Comparison of Methods

本プロジェクトでは、以下の方法で推定した介入効果を比較します。

| Method                        | Description     | Estimated ATE | Notes             |
| ----------------------------- | --------------- | ------------: | ----------------- |
| Naive Difference              | 調整なしの平均アウトカム差   |               | 交絡を調整していない        |
| Propensity Score Matching     | 傾向スコアが近い個体同士を比較 |               | マッチング後の共変量バランスが重要 |
| Inverse Probability Weighting | 傾向スコアに基づいて重みづけ  |               | 極端な重みに注意          |
| Linear Regression Adjustment  | 回帰モデルで共変量を調整    |               | モデル指定に依存          |

目的は、手法ごとの推定結果が一貫しているかを確認することです。

推定値が大きく異なる場合は、以下のような原因を確認します。

1. 傾向スコアモデルが適切でない
2. 共通サポートが十分でない
3. IPWの重みが極端に大きい
4. 共変量バランスが十分に改善していない
5. 回帰モデルの指定が適切でない
6. マッチング後にサンプルサイズが大きく減っている

## 8. Key Takeaways

本プロジェクトから得られる主な学びは以下です。

* 介入群と非介入群を単純比較すると、介入割り当てに関係する交絡因子の影響を受ける可能性がある
* 傾向スコアマッチングでは、介入を受ける確率が近い個体同士を比較する
* 逆確率重みづけ推定では、介入確率の逆数を使って母集団を調整する
* 線形回帰による調整では、観測された共変量をコントロールしたうえで介入効果を推定する
* 各手法は異なる仮定に依存するため、推定結果を比較することが重要である
* 因果効果として解釈するためには、未観測交絡がないことや正値性などの仮定が必要である

このプロジェクトの目的は、単に1つの介入効果を推定することではありません。

複数の因果推論手法が、どのように交絡を調整し、どのように推定結果が変わるのかを理解することを目的としています。

---

## 9. Assumptions

推定結果を因果効果として解釈するためには、以下の仮定が必要です。

### 1. Consistency

一致性とは、実際に観測されたアウトカムが、実際に受けた処置に対応する潜在アウトカムと一致する、という仮定です。

```text
T = 1 の場合、Y = Y(1)
T = 0 の場合、Y = Y(0)
```

### 2. Conditional Exchangeability / Strong Ignorability

条件付き交換可能性、または強い無視可能性とは、観測された共変量 `X` で条件づければ、処置の割り当て `T` が潜在アウトカムと独立である、という仮定です。

```text
(Y(1), Y(0)) ⫫ T | X
```

これは、同じような特徴を持つ個体同士で比較すれば、処置の有無はランダムに近いものとして扱える、という意味です。

### 3. Positivity

正値性とは、観測された共変量 `X` の各パターンにおいて、介入を受ける確率も、受けない確率も正である、という仮定です。

```text
0 < P(T = 1 | X) < 1
```

これは、各タイプの個体について、介入群と非介入群の両方が存在する必要がある、という意味です。

### 4. SUTVA

SUTVAは、Stable Unit Treatment Value Assumption の略です。

これは、主に以下の2つを意味します。

1. ある個体への介入が、別の個体のアウトカムに影響しない
2. 介入の内容が明確に定義されており、複数の異なるバージョンがない

これらの仮定が成り立つとき、交絡調整後の推定値を因果効果として解釈できます。
